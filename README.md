📋 Lista de Tarefas - Next.js

📌 Sobre o projeto
Aplicação de lista de tarefas desenvolvida com Next.js, utilizando React e TypeScript. Permite adicionar tarefas e visualizar a quantidade total cadastrada em tempo real. O projeto foi criado com foco em prática de conceitos fundamentais de desenvolvimento front-end moderno.

🚀 Tecnologias utilizadas
Next.js
React
TypeScript
Jest
Testing Library

⚙️ Funcionalidades
Adicionar novas tarefas
Listar tarefas cadastradas
Contador de tarefas em tempo real

📂 Estrutura do projeto
app/page.tsx → Página principal
components/NovaTarefa.tsx → Componente de input
hooks/useContadorDeTarefas.ts → Hook de contagem
lib/tarefas.ts → Dados iniciais
tests/ → Testes automatizados

🧪 Testes
Para rodar os testes:

npm run test

▶️ Como rodar o projeto
Instale as dependências:

npm install

Execute o projeto:

npm run dev

Acesse no navegador:
http://localhost:3000

Este projeto foi desenvolvido com foco em aprendizado e prática de conceitos como componentização, criação de hooks personalizados e testes automatizados.
